# 步骤5：向量化（`vectorize`）

对应实现：`knowledge_graph/agents/vectorization.py`、`knowledge_graph/utils/vector_db.py`

## 架构流程图

```mermaid
flowchart TD
    A[读取 state.knowledge_points] --> B[组装待向量化实体]
    B --> C[初始化 VectorDBManager(force_recreate=true)]
    C --> D[批量embedding写入向量库]
    D --> E[标记 state.vector_db_updated=true]
```

## 详细实现说明

- **输入**
  - `state.knowledge_points`
- **核心逻辑**
  - 当前实现 **仅向量化知识点**（`state.knowledge_points`），资源不参与向量化以节省成本。
  - 采用向量库管理器 `VectorDBManager(force_recreate=True)` 完成存储更新。
- **输出**
  - `state.vector_db_updated = True`
- **异常与边界**
  - 若无实体可向量化，会记录 warning 并写入 `state.errors`。
  - 向量化失败由异常捕获后返回主流程处理。
